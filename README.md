📌 FLAN – Landing Page para Agência de Serviços Digitais

Projeto de landing page institucional desenvolvido para uma agência fictícia de serviços digitais chamada FLAN.
O site apresenta os principais serviços da empresa, valores institucionais e uma área de contato para potenciais clientes.

A página foi construída com foco em design moderno, navegação simples e experiência do usuário, utilizando animações suaves e layout responsivo.

🚀 Funcionalidades

Header com menu de navegação

Seção principal (hero section) com chamada para ação

Seções institucionais (Missão, Valores e Visão)

Apresentação de serviços com efeito flip card

Área de clientes/parceiros

Seção de mapa integrada com Google Maps

Formulário de contato

Animações de scroll usando AOS - Animate On Scroll Library

Footer com redes sociais

Atualização automática do ano no copyright com JavaScript

🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript

AOS - Animate On Scroll Library

Google Maps (embed)

🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar desenvolvimento front-end, estruturação de landing pages e aplicação de conceitos de UI/UX para sites institucionais.
